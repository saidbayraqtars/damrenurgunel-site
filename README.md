# Samsun Klinik Psikolog Damrenur Günel - Resmi Web Sitesi

Modern web teknolojileri (Next.js 15, React 19, Tailwind CSS) kullanılarak, "Samsun Psikolog" ve "Samsun Klinik Psikolog" aramalarında üst sıralamalara çıkmak hedefiyle SEO uyumlu olarak tasarlanmış olan resmi web sitesidir.

## Proje Hakkında

Bu proje, **Klinik Psikolog Damrenur Günel**'in dijital varlığını sağlamak ve danışanlarının ona kolayca ulaşabilmesini mümkün kılmak için geliştirilmiştir. Site üzerinden randevu alınabilir, klinik iletişim detaylarına ulaşılabilir ve verilen hizmet profilleri (bireysel terapi, çocuk-ergen terapisi, aile-çift terapisi vb.) incelenebilir.

### Öne Çıkan Özellikler

- 🚀 **Next.js & App Router:** Gezinmede yüksek performans ve esnek yönlendirme mimarisi.
- 🎨 **Tailwind CSS & Remix Icons:** Modern, duyarlı (responsive) tasarım ve profesyonel arayüz bileşenleri.
- 🌐 **Kapsamlı SEO (Samsun Odaklı):** Bütün sayfalar için "samsun klinik psikolog, samsun psikolog, samsun çift terapisi" kelime grupları gözetilerek yazılmış meta etiketleri ve On-Page SEO iyileştirmeleri.
- 📱 **Mobil Uyumluluk:** Responsive tasarım standartlarına tamamen uygun mobil arayüz.
- 💡 **Dinamik Metadata Yönetimi:** Her hizmet ve blog için Next.js `metadata.ts` üzerinden yapılandırılmış gelişmiş OpenGraph ve Keyword yapısı.

## Teknoloji Yığını

- **Framework:** [Next.js 15.1.4](https://nextjs.org/)
- **Kütüphane:** [React 19](https://react.dev/)
- **Stil Yönetimi:** [Tailwind CSS 3.4](https://tailwindcss.com/)
- **İkonlar:** [Remix Icon 4.5](https://remixicon.com/)
- **Dil:** TypeScript

## Kurulum ve Çalıştırma

Geliştirme ortamınızda projeyi ayağa kaldırmak için aşağıdaki adımları kullanabilirsiniz:

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

3. Tarayıcınızda `http://localhost:3000` adresine gidin.

### Build & Production
Prodüksiyon build'ini almak için:
```bash
npm run build
npm run start
```

## SEO Politikası
- Projede `app/layout.tsx` ve diğer sayfa ile hizmet alt dizinlerindeki `metadata.ts` dosyaları tamamen bölge hedefli SEO (Samsun, İlkadım, Atakum) kurallarına göre güncellenmiştir.
- `robots.txt` ve `sitemap.xml` public dizininde yapılandırılmıştır, arama motorları için uygundur.
